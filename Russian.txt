Нужно написать простое web-приложение, реализующее продолжительный по времени выполнения REST сервис.

Исходные данные - текстовый файл со списком имен и днями рождений (в произвольном, удобном для кандидата формате). Функционально сервис должен принимать один необязательный параметр - месяц в числовом виде (от 1 до 12) - и возвращать список имен, имеющих дни рождения в указанном месяце с информацией о том, сколько дней осталось до дня рождения каждого. Если параметр не задан, то по-умолчанию должен быть использован текущий месяц. Результат должен быть возвращен в формате JSON. Сервис должен эмулировать продолжительный по времени процесс (скажем 1-2 минуты).

Сервис должен реализовывать два запроса: постановку задачи на выполнение и запрос на получение результата. В первом случае передается необязательный параметр (месяц) и возвращается идентификатор процесса, которые потом используется в качестве параметра для второго запроса. Второй запрос должен вернуть либо результат, либо сообщение о том, что задача еще выполняется.

Архитектурно сервис должен быть реализован на базе spring framework. Обязательно логирование операций с применением logging framework по выбору кандидата. Для сборки приложения должен быть написан простой скрипт (допускается использование любой системы сборки по выбору кандидата, использование скриптов автоматически созданных средой разработки запрещено). Приложение должно быть готово к запуску в любом J2EE совместимом сервлет-контейнере.

Разрешено применение любых дополнительных библиотек/пакетов по желанию кандидата. Версия Java: 7 и выше (8 - разрешено и приветствуется).
